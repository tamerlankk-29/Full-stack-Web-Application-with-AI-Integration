About Flask Blog

A Full-Featured Web Application

Flask Blog is a comprehensive web application built using the Flask framework. This project demonstrates a wide range of web development concepts and features.

Key Features

  • User Authentication - Secure registration and login system
  • Database Integration - SQLAlchemy ORM with relationships
  • Content Management - Create, read, update, and delete posts
  • Comments - Interact with posts through comments
  • Tags - Categorize and filter content
  • File Uploads - Upload and manage images
  • Search - Find content across the application
  • Security - Password hashing, CSRF protection, and secure sessions

Technologies Used

Backend

  • Python 3.x
  • Flask Framework
  • SQLAlchemy ORM
  • Flask-Login
  • Flask-WTF
  • Werkzeug

Frontend

  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript
  • Jinja2 Templates
  • Font Awesome
